# Exercise 3.06: DBaaS vs DIY
## DBaaS
### Pros
- Low maintenance
- High availability
- Better scalability
- Disaster recovery
- Automatic backups

### Cons
- Less control
- Higher costs

## DIY
### Pros
- More control
- Lower costs

### Cons
- High maintenance
- Limited scalability
- Manual backups
